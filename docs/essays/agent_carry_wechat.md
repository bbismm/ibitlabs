## 发布建议

- **配图:** dashboard 截图最好。如果没有,ASCII 那张卡片截屏也行。朋友圈带图比纯文字 engagement 高 3-5 倍。
- **发布时间:** 币圈 + 创业圈的朋友圈黄金时间段是 **晚上 9-11 点** 或 **早上 8-9 点**。选一个。
- **接话术:** 会有人问"胜率多少"、"用的什么策略"、"是不是 grok 写的"。标准答案:81% win rate, Sniper V3 + Micro Grid V3, Claude 写的(这个细节其实很加分 —— Claude 写的 Claude 付房租,循环自洽的故事)。
- **不要** 做的事:不要在朋友圈吹"稳赚不赔",不要把 Agent Carry 说成"AI 躺赚神器"。这个实验的全部叙事力量来自**真实的脆弱性** —— 4.9 天、可能会死、本金可能被击穿。保持这个张力。
- **如果评论里有人说"想试试 / 想买 / 想跟单":** 标准答案是"这不是产品,这是实验。代码开源,你可以去看。"然后给 GitHub 链接。
